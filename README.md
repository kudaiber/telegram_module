# Учебный Telegram-бот (College Support Bot)

Телеграм-бот для автоматизации заявок студентов и административного управления. Разработан для сдачи модуля (3 курс).

## Функционал

**Пользователь:**
- Регистрация (Имя, Группа, Телефон) с валидацией данных.
- Создание заявок (Текст + Фото).
- Просмотр профиля.

**Администратор:**
- Просмотр списка пользователей.
- Получение уведомлений о новых заявках.
- Просмотр списка заявок.

## Технологический стек

- **Язык:** Python 3.10+
- **Фреймворк:** aiogram 3.x (асинхронный)
- **База данных:** SQLite + SQLAlchemy (ORM)
- **Драйвер БД:** aiosqlite
- **Конфигурация:** python-dotenv
